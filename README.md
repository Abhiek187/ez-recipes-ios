Easy recipes finder iOS app

## Fastlane

Follow the [docs](https://docs.fastlane.tools/getting-started/ios/setup/) to setup Fastlane on iOS.

```bash
cd EZ\ Recipes
bundle config set --local path 'vendor/bundle'
bundle install
bundle exec fastlane ios tests
```
